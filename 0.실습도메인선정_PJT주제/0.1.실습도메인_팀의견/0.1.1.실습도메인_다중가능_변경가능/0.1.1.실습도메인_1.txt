
[결혼해조 :: 땡큐 메리 마치]

1. 사이트 명 : 
	땡큐 메리 마치

2. 주제 : 	
	스튜디오 예약 및 드레스 구입 등, 결혼 준비 과정을 한 사이트에서 비교, 파악할 수 있는 웨딩 준비 플랫폼

3. 캐치프라이즈 : 
	쇼핑처럼 쉬운 결혼 준비, 내 맘대로 견적내고 예약하고 구매까지

4. 기존 문제점 :
    - 플래너에게 의존한 웨딩 플랜
    - 가격이 비쌈
    - 웨딩 플래너와 협력하는 업체가 아닌 다른 업체를 이용하고 싶을 시, 직접 발로 뛰어야 함
    - 가격의 불투명성

5. 목표
    - 웨딩 플래너의 역할을 대신하는 웹사이트
    - B2B : 웨딩과 관련된 업체, 개인 사업자끼리의 연결
    - B2C : 합리적인 단가의 웨딩 정보를 필요로 하는 소비자와 업체, 혹은 개인 사업가와의 연결

6. END USER 입장에서 바라는 기능

    6.1. 유저별 관리 시스템 : 예비 배우자와 함께 공유 가능
        - 일정 관리
        - 체크리스트 : 일정 세분화 및 업체 선정 고려사항 선제공
        - 예산 관리
        - 예약

    6.2. 셀프 플래닝
        - 스+드+메 묶음이 아닌, 필요로 하는 과정만 장바구니에 담아 견적을 확인할 수 있음
    
    6.3 결제
        - 카드, 제휴 결제, 무통장 입금 등 다양한 결제 방식
    
    6.4 검색
        - 해시태그 : 웨딩홀, 드레스 등의 광범위한 컨셉들을 분류, 맞춤 검색, 직관적인 후기
        - 스타일 순위 검색 및 추천 : 다수의 유저들이 찜한 스타일 확인 가능
        - 가격 비교 : between

    6.5 한눈에 파악할 수 있는 카테고리 : 지역별 / 기능별(스튜디오, 메이크업샵, 드레스샵, 청첩장 별 등)

    6.6 커뮤니티

    6.7 투명한 가격 공개 및 업체 정보
        - 웨딩 견적 맵 : 원하는 지역의 견적 정보를 지도로
        - 업체별 포트폴리오 블로그 (ex.인스타그램 피드) + 실사용자들의 리뷰 & 추천, 비추천
        - 상담 게시판, 채팅 기능

    6.8 이벤트
        - 업체와 contact 후, 할인쿠폰 제공
        - 회원가입시 추천코드
        - 후기 등록 시 포인트 제공
        - (신혼여행)

7. 기대효과
    - 직접 발품을 팔아 업체를 찾아야하는 시간적인 측면과, 비용적인 측면에서 사용자의 부담을 줄일 수 있다.
    - 해당 데이터들을 모으면 여러 업체에 판매할 수 있으며, 광고를 넣을 수 있다.
