1. 프로젝트 주제명
반려동물 모임

2. 선정이유

반려동물 보유 가구 수 증가
초보집사들을 위한 정보 공유
신뢰성 있는 입양

3. 주요기능

 - 동물병원 / 호텔 / 여가 장소 지도표시
 - 회원가입 및 반려묘/반려견 정보 등록
 - 이용자 지역/최대인원수/상세 정보 입력해서 모임등록
 - 모임기간 결제
 - 모임 유지기간 / 인원수 별 등급 나뉨 / 개인 별 등급
 - 등급 별 이점 ( 상위 노출, 모임 비용 할인 ) 
 - 해시태그 / 즐겨찾기 / 알림
 - 채팅 / 쪽지
 - 모임 날짜 시간 및 장소 등록 / 신청
 - 입양/중고물품 관련 글 게시판

4.기대효과

모임을 통해 반려동물 관련 정보 공유로 동물 보호

5. 고려해야할 사항 및 특이기술

지도 API , 결제 시스템
