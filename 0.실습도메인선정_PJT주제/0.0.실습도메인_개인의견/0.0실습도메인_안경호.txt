1. project주제명. 여행 가이드 사이트 (내 / 외국인(주타겟))
2.. 선정이유
  1. 현재 한국은 넷플릭스, BTS로 많이 알려지고  코로나가 잠잠해 지면 한국에 외국인 관광객이 많이 방문할 가능성이 높다는 뉴스기사
  2. 현재 대표적인 플랫폼은 트리플이 있다. 주로 서울 인근지역으로 서비스가 집중되어 있고 세세한 여행경로를 지원해주지 않는다
3. 주요기능
    - 예약 시스템(호텔, 자동차,식당) 
	외국인들이 이용할 수있는 번역된 예약 시스템
    - 결제 시스템 (보안문제)
    - 회원가입
    - 이동
	택시 / 버스 / 지하철등 어떻게 이동수단을 이용해야하는지 (카카오맵 길찾기)
    - 번역 서비스 
	외국인 맞춤 번역 서비스
    - 후기등록
	인스타와 같이 해당 여행 경로를 이용한 사람들의 사진과 후기를 남길 수 있는 서비스 / 추천 여행지
 4.기대효과
   - 서울지역외에 넷플릭스 촬영지 / 연예인 출생지등을 여행지로 추천하고 지역경제를 올리는 효과
