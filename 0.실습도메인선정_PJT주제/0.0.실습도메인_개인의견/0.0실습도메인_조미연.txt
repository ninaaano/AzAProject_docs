1. 프로젝트 주제명 
ERP
- 전사적자원관리 : 비즈니스 전반에 걸친 업무 프로세스를 통합적으로 관리할 수 있도록 돕는 시스템

2. 선정이유 
이카운트, 하이웍스 벤치마킹

3. 주요 기능
- 직원관리
  > 휴가, 급여, 근태 관리 등
- 거래처 관리
  > 거래처와 소통할 수 있는 게시판.
  > 거래처 담당자 정보
- 스케줄 관리
  > 같은 부서와 공유 가능
- 전자 결재 기능
- 채팅 기능
  > 직원간 채팅 기능
- 메일 보내기
- 계산서 발행

4. 기대효과
- 페이지 내에서 모든 업무 관리, 업무 수행 가능.

5. 기술적으로 고려할 사항 및 특이기술 
메일 수/발신, 계산서 발행 구현시 다른 서버와 통신(?)해야하는 기능, 암호화 기능
주소록, 매출 내역 등 엑셀로 일괄 업로드 기능.
